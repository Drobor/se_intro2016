VectorF
    вектор. 32 битная точность вычислений. NaN +-inf являются допустимыми значениями

    public VectorF(final int size)
        создвет вектор с заданной размерностью size
        throws
            Вызывает исключение, если size отрицателен

    public void set(final int pos, final float val)
        записывает значение val в вектор под индексом pos
        throws
            Вызывает исключение, если индекс вектора превышент размерность или отрицателен

    public float get(final int pos)
        возвращает значение вектора под индексом pos
        throws
            Вызывает исключение, если индекс вектора превышент размерность или отрицателен

    public int getSize()
        возвращает размерность вектора

    public VectorF add(final VectorF vector)
        возвращает вектор элементы которого равны сумме элементов текущего экземпляра и vector с одинаковыми индексами
        throws
            Вызывает исключение, если размерность текущего экземпляра не совпадает с размерностью vector

    public VectorF sub(final VectorF vector)
        возвращает вектор элементы которого равны разности элементов текущего экземпляра и vector с одинаковыми индексами
        throws
            Вызывает исключение, если размерность текущего экземпляра не совпадает с размерностью vector

    public VectorF mul(final VectorF vector)
        возвращает вектор элементы которого равны произведению элементов текущего экземпляра и vector с одинаковыми индексами
        throws
            Вызывает исключение, если размерность текущего экземпляра не совпадает с размерностью vector

    public VectorF div(final VectorF vector)
        возвращает вектор элементы которого равны частному элементов текущего экземпляра и vector с одинаковыми индексами
        throws
            Вызывает исключение, если размерность текущего экземпляра не совпадает с размерностью vector

    public VectorF mul(final float val)
         возвращает вектор элементы которого равны произведению элементов текущего экземпляра и val

    public VectorF div(final float val)
         возвращает вектор элементы которого равны частному элементов текущего экземпляра и val

    public float length()
        врзвращает длину вектора

    public VectorF normal()
        возаращает нормализованный вектор

    public float mulScalar(final VectorF vector)
        возвращает результат скалярного произаедения векторов
        throws
            Вызывает исключение, если размерность текущего экземпляра не совпадает с размерностью vector

    public static VectorF mulVector(final VectorF... vector)
        возвращает векторное произведение векторов vector. В случае, если размерность аектора не равна трём под векторным произведением понимается детерминант матрицы, в которой первая строчка заполнена базимными векторами, а последующие n-1 строчек заполнены векторами vector, являющимися сомножитклями векторного произведения
        throws
            Вызывает исключение, если размерность элементов vector не совпадает
            Вызвыает исключение, если количество векторов не равняется из размерности -1 вектор


SquareMatrixF
    квадратная матрица. 32 битная точность вычислений. NaN +-inf являются допустимыми значениями

    public SquareMatrixF(final int size)
        создвет квадратную матрицу с заданной размерностью size
        throws
            Вызывает исключение, если size отрицателен

    public void set(final int x, final int y, final float val) {
        записывает val а ячейку [x][y]
        throws
            Вызывает исключение, если x или y меньше 0 или превышают размерв матрицы

    public float get(final int x, final int y)
        возвращает значение ячейки [x][y]
        throws
            Вызывает исключение, если x или y меньше 0 или превышают размерв матрицы

    public float determinant()
        возвращает определитель матрицы
